# Hướng dẫn chạy app KMA Legend

### Bước 1: Truy cập vào github và clone dự án về máy hiện tại
`$ git clone https://github.com/mrcodewalker/desktop-app.git
$ cd desktop-app/KMALegendDesktop`

### Bước 2: Khởi chạy chương trình bằng dòng lệnh
`$ mvn javafx:run`
